# **TFK ShadoW - Portfolio Website**

## **Description**
This is a responsive portfolio website showcasing my skills, projects, and experience in frontend development. The website features a dark mode/light mode toggle, animated elements, and an intuitive navigation system for an optimal user experience.

## **Features**
- **Responsive Design** - Fully functional across all screen sizes.
- **Dark Mode & Light Mode** - Users can toggle between themes for better readability.
- **Smooth Navigation** - Links to different sections for an easy browsing experience.
- **Projects Showcase** - Displays projects with GitHub and live demo links.
- **Contact Section** - Users can quickly find ways to connect via email or LinkedIn.

## **Technologies Used**
- **HTML5** - Structure and content.
- **CSS3 (SCSS)** - Styling, animations, and responsive design.
- **JavaScript (ES6+)** - Functionality for theme toggling and interactive elements.
- **Google Fonts** - Typography customization.

## **Folder Structure**
```
/root
│── index.html              # Main HTML file
│── styles.css              # General styles
│── mediaqueries.css        # Responsive design rules
│── dark-mode.css           # Dark mode specific styles
│── light-mode.css          # Light mode specific styles
│── index.js                # JavaScript for functionality
│── images/                 # All project and profile images
```

## **Setup & Installation**
1. Clone the repository or download the files.
   ```
   git clone https://github.com/yourgithubusername/portfolio-website.git
   ```
2. Open `index.html` in a browser.
3. To modify styles, edit the respective CSS files.
4. For JavaScript functionality, modify `index.js`.
